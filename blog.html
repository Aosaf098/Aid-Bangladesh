Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is the hierarchical data representation of HTML documents which defines the structure and views the webpage elements as a tree of nodes. It allows the Javascript to manipulate and modify contents dynamically

How do you select an element from the DOM?

You can select an element from the DOM by using multiple ways. You can use getElementById(), getElementsByClassName(), querySelector() and querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is an event handling procedure by which we attach a single event listener to a parent element instead of attaching event listeners to every child element. It is useful because it can ensure better performance when dealing with dynamic websites with so much less code

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element’s attributes by using methods like getAttribute(), setAttribute(), hasAttribute(), removeAttribute(). We can manipulate an element’s styles by using the style property
